# 🍎 Macro Food Tracker
An AI-powered nutrition assistant that analyzes food items and tracks daily macros.

## 🚀 Live Demo
Check out the current build here: [View App Preview](https://calorie-counter-557.preview.emergentagent.com/)

## ✨ Features
* **AI Analysis:** Input food names and weights to get instant nutritional data.
* **Macro Tracking:** Real-time summary of Calories, Protein, Carbs, and Fats.
* **Modern UI:** Clean, dark-mode interface built for mobile users.

## 🛠️ Tech Stack
* **Frontend:** React Native / TypeScript (Expo)
* **Backend:** Python (FastAPI)
* **Intelligence:** Integrated AI for food recognition
